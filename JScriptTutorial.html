Learn How To Program in JavaScript

What is JavaScript?

  • Scripting, Interpreted Language (Not compiled. Belongs to the same class of Languages as Python)
  • Conforms to ECMAScript specifications
  • Multi-Paradigm (Object Oriented, Functional, etc)
  • Runs on client/browser as well as on the server(Node.js)
    • JavaScript Manipulates the Browser as JavaScript (It's the language for manipulating Browsers/ language for developing apps for Browsers)
    • JavaScript Manipulates the Server as Node.JS. (Node.JS is a JavaScript runtime)

Uses of JavaScript?

  • Program the Browser
  • Build Server and Full Stack Applications
  • Build Interactive User Interfaces with frameworks like:
    • React Native
    • NativeScript
    • Ionic
  • Build Mobile Applications with frameworks like:
    • ReactJS
    • AngularJS
    • ViewJS
  • Build Desktop Applications with frameworks like:
    • ElectronJS

    • Note: JavaScript interacts with HTML through DOM, therefore JavaScript files will not understand instructions referencing HTML element tags directly.
    The Console in the Browsers Developer Tools, acts as and virtual machine or IDE running JavaScript commands that are displayed by the the Browser or alternatively on the console using ( console.log())
